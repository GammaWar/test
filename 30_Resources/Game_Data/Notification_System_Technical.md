# Benachrichtigungssystem (Notification System) & Kurzbeschreibung
Das Spacewar-Benachrichtigungssystem ermöglicht es Spielern, über wichtige Ereignisse im Spiel informiert zu werden. Seit Version 0.5.129 ist dieses System hochgradig granular steuerbar.

## Basisdaten

| Kategorie | Beschreibung |
| :--- | :--- |
| **Recycling** | Benachrichtigungen über abgeschlossene oder gestartete Recycling-Prozesse. |
| **Spionage** | Hinweise auf erfolgreiche oder gescheiterte Spionageversuche. |
| **Überantrieb** | Meldungen bezüglich der Nutzung von Überantriebs-Modulen/Fähigkeiten. |
| **Kampfberichte** | Benachrichtigungen über neue Kampfberichte (Battle Reports). |
| **Support-Antworten** | Hinweise auf Antworten aus dem Support-System. |

## Mechanik / Verhalten

### Granulare Steuerung
Spieler können in den Einstellungen festlegen, für welche Ereignistypen sie Benachrichtigungen erhalten möchten. Dies dient der Vermeidung von Informationsüberflutung und ermöglicht eine Fokussierung auf die für den eigenen Spielstil (z. B. Miner vs. Fleeter) relevanten Events.
