# Das Erwachen von Vastor-7 & Kurzbeschreibung
Ein historisches Lore-Ereignis auf dem Waldmond Vastor-7, das die Ankunft der Legion Space Division markierte.

## Basisdaten

| Aspekt | Details |
| :--- | :--- |
| **Ort** | Vastor-7 (Waldmond) |
| **Beteiligte Fraktion** | Legion Space Division (Aegis-Prime) |
| **Ereignistyp** | Mysteriöses Energieereignis / Ankunft eines Kolonieschiffs |

## Mechanik / Verhalten

### Hintergrund
Vastor-7 war ursprünglich ein stiller, unbedeutender Waldmond, bedeckt von dichten Tannenwäldern. Die Stille des Mondes wurde durch ein mysteriöses Ereignis jäh unterbrochen.

### Das Ereignis
In einer Nacht schoss ein weiß-blauer Energiestrahl aus den Wolken und traf ein kleines Haus inmitten der Wälder. Der Himmel glühte und bebte, während ein gigantisches Kolonieschiff lautlos über dem Ort schwebte. 

Als die **Aegis-Prime** der **Legion Space Division** eintraf, war das fremde Schiff bereits verschwunden. Zurück blieb lediglich ein Krater, ein zerstörtes Haus und ein rätselhaftes Objekt im Zentrum des Einschlags, das dort scheinbar seit Jahrtausenden gewartet hatte.
