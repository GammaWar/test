# Mobile Raumstationen (Konzept) & Kurzbeschreibung
Konzept für die Nutzung von Raumstationen als mobile Bergbau-Einheiten zur Ressourcenextraktion auf unbesetzten Planeten.

## Basisdaten

| Eigenschaft | Beschreibung |
| :--- | :--- |
| **Hauptfunktion** | Ressourcenabbau auf leeren Planeten |
| **Mobilität** | Stationen können zu Planeten geschickt werden (langsame Bewegung) |
| **Risiko** | Hohe Angreifbarkeit durch andere Spieler/NPCs |

## Mechanik / Verhalten

Raumstationen könnten mit einem speziellen Modul ausgestattet werden, das es ermöglicht, sie als mobile Basis zu nutzen. 

1. **Einsatz:** Eine Station wird zu einem unbesetzten Planeten geschickt.
2. **Bergbau:** Die Station führt dort den Ressourcenabbau durch.
3. **Verteidigung:** Da die Station mobil und potenziell isoliert ist, muss sie aktiv verteidigt werden, da sie sonst zerstört werden kann.
